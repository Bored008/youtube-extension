# YouTube Downloader Extension

A Chrome extension built to learn backend development concepts through a real-world project.

This project allows users to download YouTube videos directly from the browser using a custom backend server. It helped in understanding APIs, request handling, streaming, file processing, browser extensions, and client-server communication.

---

## 🚀 Features

* Download YouTube videos from the browser
* Chrome extension integration
* Backend API handling
* Video stream processing
* Simple and clean UI
* Real-time request flow between frontend and backend
* Local server communication

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Bored008/youtube-extension.git
```

### 2. Open the Project

```bash
cd youtube-extension
```

### 3. Install Backend Dependencies

```bash
npm install
```

### 4. Start the Backend Server

```bash
node server.js
```

or

```bash
npm start
```

---

## 🧩 Load Chrome Extension

1. Open Chrome
2. Go to:

```text
chrome://extensions/
```

3. Enable **Developer Mode**
4. Click **Load Unpacked**
5. Select the extension folder
